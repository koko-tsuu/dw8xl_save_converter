
# Dynasty Warriors 8 Xtreme Legends Save Converter
A program that allows you to convert your Dynasty Warriors 8 Xtreme Legends save to different platforms (PC, PS3, PS4, PSV). 

    
## Download
Download latest release [here](https://github.com/koko-tsuu/dw8xl_save_converter/releases/latest), then unzip.

## Usage
- Input
    - Click on browse, and navigate and select the file you want to convert.
    - Note that when you select a PS3, PS4, and PSV savefile, they must be unencrypted.
- Output
    - Navigate to the directory you want the program to save to, then add both the filename and the extension (ex. for PC, "save.dat" without the quotations). You may want to name the file to the platform's desired save file name to easily copy and paste.
- Input Type
    - Select the platform that the input file comes from.
- Output Type
    - Select the platform that you want to convert the save file to.
    - When converting a file for PS3, PS4, PSV, it will also be unencrypted, meaning you must encrypt/sign it with your PSN account.


## Save Filenames Reference
- PC: save.dat
- PS3: APP.BIN
- PS4: APP.BIN
- PSV: MainData00.dat
